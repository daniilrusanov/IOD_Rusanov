# Експертне опитування — ЛР1 та ЛР2 ІОД РІС

Веб-додаток для проведення попереднього преференційного голосування та евристичного звуження підмножини об'єктів.

## Зміст завдань

**Лабораторна 1** — розподілене введення даних:
- 20 об'єктів, 20+ експертів
- Пріоритетний вибір 3 об'єктів (множинне порівняння)
- Анонімне опитування з конфіденційністю
- Збереження даних для подальшої обробки

**Лабораторна 2** — розподілена обробка даних:
- Підмножина об'єктів з результатів ЛР1 (>10, <20)
- Вибір 2–3 евристик для звуження до макс. 10 об'єктів
- Відкрите опитування з протоколом
- Евристики та еволюційний алгоритм для ранжування

## Функціонал

- **ЛР1**: Голосування — вказати ім'я, обрати 3 об'єкти з 20
- **ЛР2**: Обрати 2–3 евристики для відсіювання об'єктів
- **Адмін**: `admin` / `admin` — протокол та результати

## Запуск

```bash
npm install
npm run dev
```
